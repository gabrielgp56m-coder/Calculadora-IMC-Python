# Calculadora de IMC em Python

Projeto desenvolvido em Python para calcular o IMC (Índice de Massa Corporal) e estimar o peso ideal com base nos dados informados pelo usuário.

## Funcionalidades

Cálculo do IMC
Classificação do resultado entre abaixo do peso, normal, sobrepeso e obeso
Estimativa de peso ideal com base em altura, idade e sexo
Tratamento básico de erros

## Tecnologias utilizadas

Python

## Como funciona

O programa solicita peso em kg, altura em cm, sexo e idade.
Ao final, retorna o valor do IMC, a classificação e o peso ideal aproximado.

## Como executar

```bash
git clone https://github.com/seu-usuario/Calculadora-IMC-Python.git
```

```bash
cd Calculadora-IMC-Python
```

